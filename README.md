# TaipeiZooApiSample

Simple project to have API connection to Taipei metadata and get Taipei zoo and plants data.

# Implementation

* Design Patterns

  * Object-Oriented Programming
  * Model-View-Presenter (MVP)
  * Model-View-ViewModel (MVVM)
  * Repository Pattern
  * UseCase Pattern
  
* User Interface

  * Fragment + TransactionAnimation
  * RecyclerView
  * DrawerToggle
  * ProgressBar
  * Picasso (Third-party Library)
  
* NetworkCheker

  * BroadcastReceiver
  
* Archetecture Component
  * ViewModel
  * LiveData
  
* API

  * Retrofit
  * RxJava2 - Thread control
  
* Unit Test
  * MockK
  * RxTest
  
# Requirement

* Android Studio 3.3.2+
* Android SDK 23+
* Gradle 3.3.2+
* Java 8 - for Lambda expression
* Support AndroidX
* Kotlin
